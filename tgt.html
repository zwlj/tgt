1. 项目概览

东京猛兽 (Tokyo Beast) 旨在构建一个全面的加密娱乐IP生态系统，其核心为东京游戏代币 (TGT)。项目以AAA级游戏品质为目标，融合Web3技术，并获得了日本知名企业的支持。

项目愿景

打造首款旗舰AAA级游戏，设定在2124年的未来东京，玩家操控AI驱动的“猛兽”机器人进行竞技。TGT代币将作为旗下多款高端区块链游戏的通用货币。

主要支持者

  • SBI集团 (大型金融机构)
  • Cygames (日本顶级游戏开发商)
  • gumi Inc. (移动游戏与Web3投资)

运营公司：TOKYO BEAST FZCO (迪拜)

游戏开发：gumi Inc.

区块链技术：Turingum K.K.

4. 核心经济机制

项目设计了多项核心经济机制以确保TGT代币的价值支撑和生态系统的可持续性，其中最引人注目的是游戏收入回购计划。

15% 游戏收入回购机制

项目承诺将所有游戏总收入的15%用于回购TGT代币，为奖励池提供资金并避免通胀。这可能为TGT创造一个“需求基石”。

竞猜机制 (即将推出)

TGT可用于购买 LUCKY TICKET NFT，锁定后获取 LUCKY CHIPs，用于对TRIALS锦标赛结果进行竞猜，赢取TGT奖励。

此多层系统可能旨在合规和增加用户参与，但也增加了复杂性。

TGT还可用于购买游戏内货币 (JEWELS, GEMS) 和物品，并赋予持有者项目治理权 (细节待公布)。

5. 游戏生态系统

东京猛兽采用独特的双轨系统设计，结合Web3资产交互与传统Web2游戏体验，并通过NFT代理模型等创新机制平衡游戏性与资产价值。

"BASE" 系统 (Web3)

专注于Web3资产交互：

  • NFTs (BEAST NFT, BEAST RAWDISK NFT, LUCKY TICKET NFT)
  • TGT 代币质押
  • 使用 LUCKY CHIPs 进行预测游戏
  • 以优惠价格购买 JEWELS

"TRIALS" 系统 (Web2)

提供传统免费游戏体验：

  • PvP / PvE 战斗
  • 通过 Gacha 获取 PROXY BEASTs
  • 通过应用商店购买 JEWELS
  • 无需NFT或钱包即可开始游戏

TGT 是连接这两个系统的核心货币。这种双轨制旨在降低Web3门槛，吸引传统玩家，逐步引导其探索Web3元素。

BEAST NFT 与 NFT代理模型

BEAST NFT是游戏核心，其元数据生成游戏内“代理猛兽(proxy BEASTs)”用于战斗。NFT持有者根据代理猛兽的使用频率和表现赚取奖励。

NFT代理模型优势：

  • 将资产所有权(NFT)与游戏内效用分离。
  • 允许开发者在不影响NFT属性的情况下平衡游戏。
  • 保护NFT价值，同时为持有者提供可持续奖励。
  • 解决NFT效用常见问题，更像“蓝图”而非直接游戏资产。

游戏内还有JEWELS（用于竞猜和应用内购买）和GEMS（用于抽奖等）两种货币。多种货币体系可能服务于特定战略目的（如合规、经济调控），但也可能增加用户复杂性，需要清晰传达其用途。

7. 优势与风险

东京猛兽项目展现出多方面优势，但也存在一些潜在风险和需要进一步明确的领域。全面了解这些因素对于评估项目前景至关重要。

主要优势 👍

  • 强大的行业背景和合作伙伴支持 (SBI, Cygames, gumi)。
  • 创新的15%游戏收入回购机制，保障经济可持续性。
  • TGT代币在质押、竞猜、购买方面具明确实用价值。
  • 独特的BASE/TRIALS双系统设计，广泛吸纳用户。
  • 初期社区关注度和参与度高 (百万预注册)。
  • 采用可扩展的Layer 2技术 (Immutable zkEVM)。
  • 高达40%代币分配给生态系统增长和奖励。

潜在风险与待明确领域 🤔

  • 团队、基金会、顾问代币的详细锁仓与释放计划缺乏公开信息 (透明度缺陷)。
  • GEMS货币的完整机制和具体效用尚不清晰。
  • 代币销毁等通缩机制细节仍模糊或仅暗示。
  • 治理模型的具体运作方式未详细阐述。
  • 项目成功高度依赖游戏产品的商业成功和创收能力。
  • 许多核心功能仍标记为“即将推出”，存在执行风险。

信息不对称（如锁仓计划未明）可能影响投资者信心。项目方需妥善处理这些信息缺口以赢得长期信任。

8. 未来展望

东京猛兽制定了分阶段的发展路线图，并规划了长期的IP扩展蓝图，旨在构建一个跨媒体的娱乐品牌，进一步巩固TGT的生态核心地位。

第一纪元：孵化期 (2025 Q1-Q3)

  • TGT 代币生成事件 (TGE) 及交易所上线 (2025年5月21日)。
  • "BASE" 系统上线。
  • NFT 铸造功能。
  • 全球 "TRIALS" 模式推出。
  • "TRIALS" 新功能增加。
  • 特别锦标赛举办。

第二纪元：蜕变期 (2025 Q3-Q4)

进入新的发展阶段，具体细节待公布，预计将基于第一纪元的成果进行扩展和深化。

长期IP扩展愿景

项目暗示未来计划向动画、时尚、IP联动以及线下活动等领域拓展。

若游戏IP成功，跨媒体扩张能开辟新收入来源，提升品牌价值，并为TGT生态系统带来更大价值积累。

总结：若东京猛兽能实现高品质游戏体验且收入回购模型有效执行，TGT有潜力成为Web3游戏领域的重要代币。但弥补透明度不足、成功执行并持续吸引用户是关键。
